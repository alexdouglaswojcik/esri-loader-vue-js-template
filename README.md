# esri-loader-vue-js-template

This project is a VueJS SPA starter template for web mapping applications. Contains the following packages via NPM:

Vue Router

Vuex

Bootstrap-Vue

esri-loader

Babel

Upon install the 'map' route contains a blank web map imported using esri-loader.


## Project setup
```
npm install
```
